# Raytracing-renderer
C++ 纯控制台 光线追踪渲染器  
C++ pure console Raytracing renderer


每次输入完#每个数据#应#回车确认#(确认后会有提示)   

各个光强之和应在[0 --1.0]（double)   
Y为高    
位置为三参数X  Y  Z(double)  
方向为三参数X  Y  Z(double)  
球心为三参数X  Y  Z(double)  
颜色为三参数R  G  B [0 -- 255](int)  
半径 正数(double)  
反射程度 [0 --1.0](double)   
闪亮程度  正数(double)
